Class FetchApiDataService

Service handling all API communications for the MyMDB application. Provides methods for user management, movie data retrieval, and favorites handling.

Remarks

This service uses Angular's HttpClient for all API communications and implements error handling and response processing. All methods return Observables and include appropriate error handling.

Example

constructor(private fetchApiData: FetchApiDataService) {
  this.fetchApiData.getAllMovies().subscribe(
    (movies) => console.log(movies)
  );
}
